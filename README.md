# CheckSecHeaders Script
##### Description
检测给出的 url 服务器是否设置了安全Headers，仅适用于Python2.\*，Python3.\* 暂时没适配

##### Options
```
-h                      展示此脚本的帮助信息
--help

-e                      show English help
--english_help

-c COOKIES              设置 cookies 到 headers 中
--cookie="COOKIES"

-d                      关闭 SSL/TLS 证书验证
--disable_ssl

--proxy="PROXY_URL"     设置一个代理, 例如: http://127.0.0.1
```
